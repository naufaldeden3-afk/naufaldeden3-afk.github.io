Apa itu Coding dan Mengapa Belajar Coding?

Coding (atau pemrograman) adalah cara kita berkomunikasi dengan komputer untuk membuat aplikasi, website, game, atau bahkan robot. Untuk pemula, coding terlihat sulit, tapi sebenarnya seperti belajar bahasa baru. Mulailah dengan dasar-dasar, dan Anda bisa membangun apa saja!

Manfaat belajar coding:

  • Meningkatkan logika berpikir dan problem-solving.
  • Membuka peluang karir di tech (developer, data analyst, dll.).
  • Membuat proyek kreatif seperti website pribadi atau app sederhana.

Website ini dirancang untuk pemula absolut: dari nol hingga bisa membuat proyek pertama. Ikuti langkah demi langkah!

Panduan Instalasi Alat Coding

Tutorial Lengkap: Dari Dasar hingga Lanjutan

Kami mulai dengan HTML (untuk web), lalu CSS (styling), JavaScript (interaktivitas), dan Python (logika).

1. HTML: Struktur Halaman Web

HTML adalah "tulang" website. Tag seperti <h1> untuk judul, <p> untuk paragraf.

<!DOCTYPE html>
<html>
<head><title>Halaman Pertama</title></head>
<body>
<h1>Selamat Datang!</h1>
<p>Ini adalah paragraf pertama saya.</p>
<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
</body>
</html>
Latihan: Buat file HTML baru, salin kode di atas, tambahkan gambar dengan <img src="gambar.jpg" alt="Deskripsi">, lalu buka di browser.

2. CSS: Membuat Website Cantik

CSS menambahkan warna, font, dan layout. Tambahkan ke HTML dengan tag <style> atau file terpisah.

body { background-color: lightblue; }
h1 { color: red; text-align: center; }
p { font-size: 18px; }
Latihan: Tambahkan CSS ke file HTML sebelumnya. Ubah warna latar belakang dan font.

3. JavaScript: Membuat Interaktivitas

JS membuat website "hidup". Contoh: tombol yang mengubah teks saat diklik.

<button onclick="ubahTeks()">Klik Saya</button>
<p id="teks">Ini teks awal.</p>
<script>
function ubahTeks() {
document.getElementById("teks").innerHTML = "Teks berubah!";
}
</script>
Latihan: Tambahkan ke file HTML. Klik tombol dan lihat perubahan.

4. Python: Logika dan Algoritma

Python cocok untuk pemula. Fokus pada variabel, loop, dan kondisi.

# Program sederhana
nama = input("Masukkan nama: ")
umur = int(input("Masukkan umur: "))
if umur >= 18:
print(f"Halo {nama}, Anda dewasa!")
else:
print(f"Halo {nama}, Anda masih muda!")

# Loop sederhana
for i in range(5):
print(f"Hitungan: {i+1}")
Latihan: Buat file .py, salin kode, jalankan di terminal. Modifikasi untuk menghitung angka genap.

Debugging: Memperbaiki Kesalahan Kode

Kesalahan umum: typo, kurung tidak tertutup, atau logika salah.

  • Baca pesan error dengan teliti.
  • Gunakan print() di Python untuk melihat nilai variabel.
  • Di browser, buka Developer Tools (F12) untuk JS/HTML.
Contoh Error: Jika kode Python gagal, pastikan indentasi benar (Python sensitif terhadap spasi).
